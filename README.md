# Blogify - A Simple CRUD Blog Application

## Overview

Blogify is a minimalistic blog application built using EJS, Bootstrap, and Express.js, allowing users to share their thoughts and engage with others effortlessly. It provides a simple and intuitive interface for blogging, reading, and commenting.

## Features

1. User Authentication

Sign up and log in securely to manage your blogs.

2. Create Blogs

Easily add new blog posts with a user-friendly interface.

3. Read Blogs

Explore blogs written by other users and stay informed.

4. Comment System

Engage with others by commenting on their blog posts.

## Tech Stack

Frontend: EJS, Bootstrap

Backend: Express.js

Database: MongoDB
